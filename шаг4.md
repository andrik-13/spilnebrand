## 4.1 Структура базы данных (Supabase)

### Таблица `products`

| Поле | Тип | Описание |
|---|---|---|
| `id` | uuid | Уникальный ID (генерируется автоматически) |
| `slug` | text | URL-имя: `tshirt-white` (уникальное) |
| `name_ua` | text | Название на украинском |
| `name_en` | text | Название на английском |
| `description_ua` | text | Описание на украинском |
| `description_en` | text | Описание на английском |
| `price` | integer | Цена в гривнах (без копеек) |
| `category` | text | `tops` / `bottoms` / `sets` |
| `sizes` | text[] | Массив: `['S', 'M']` |
| `composition_ua` | text | Состав ткани (UA) |
| `composition_en` | text | Состав ткани (EN) |
| `is_active` | boolean | Показывать на сайте или нет |
| `is_new` | boolean | Новинка (показывать на главной) |
| `created_at` | timestamp | Дата добавления |

### Таблица `product_images`

| Поле | Тип | Описание |
|---|---|---|
| `id` | uuid | Уникальный ID |
| `product_id` | uuid | Ссылка на товар |
| `url` | text | URL фото в Supabase Storage |
| `position` | integer | Порядок отображения (0 = главное) |

## 4.2 Supabase Storage

Хранилище для фото товаров.

**Структура папок:**
```
bucket: product-images
  /tshirt-white/
    main.jpg         ← главное фото (position: 0)
    angle-2.jpg
    flat.jpg
  /skirt-beige/
    main.jpg
    angle-2.jpg
```

**Настройки bucket:**
- Public (фото доступны по прямой ссылке без авторизации)
- Формат: JPG или WebP
- Максимальный размер: 5MB на фото

**URL фото будет выглядеть так:**
```
https://xxxx.supabase.co/storage/v1/object/public/product-images/tshirt-white/main.jpg
```

## 4.3 API Endpoints

### GET /api/products
Возвращает список всех активных товаров.

```typescript
// Параметры запроса:
?category=tops     // фильтр по категории
?is_new=true       // только новинки (для главной)
?limit=4           // ограничение количества

// Ответ:
{
  products: [
    {
      id: "uuid",
      slug: "tshirt-white",
      name: "Футболка базова",   // ua или en в зависимости от locale
      price: 890,
      category: "tops",
      sizes: ["S", "M"],
      mainImage: "https://..."
    }
  ]
}
```

### GET /api/products/[slug]
Возвращает один товар по slug.

```typescript
// Ответ:
{
  product: {
    id: "uuid",
    slug: "tshirt-white",
    name: "Футболка базова",
    description: "Базова футболка з...",
    price: 890,
    category: "tops",
    sizes: ["S", "M"],
    composition: "100% бавовна",
    images: [
      "https://.../main.jpg",
      "https://.../angle-2.jpg",
      "https://.../flat.jpg"
    ]
  }
}
```

## 4.4 Admin-панель

**URL:** `/admin` (защищена простым паролем)

Простой интерфейс для управления товарами — без лишних сложностей.

### Страница `/admin`
```
Список товаров:
┌─────────────────────────────────────────┐
│ [Фото] Футболка базова   890₴  ✓ Активен │ [Редактировать]
│ [Фото] Спідниця бежева  1200₴  ✓ Активен │ [Редактировать]
│ [Фото] Штани прямі      1500₴  ✗ Скрыт   │ [Редактировать]
└─────────────────────────────────────────┘
[+ Добавить товар]
```

### Страница `/admin/products/new` и `/admin/products/[id]`

Форма добавления/редактирования товара:

```
Название (UA): [________________]
Название (EN): [________________]
Slug:          [________________]  (автозаполняется из названия)
Категория:     [Tops ▼]
Цена (₴):      [________________]
Размеры:       [✓ S] [✓ M]

Описание (UA): [________________]
               [________________]

Описание (EN): [________________]
               [________________]

Состав (UA):   [________________]
Состав (EN):   [________________]

Новинка:       [✓]
Активен:       [✓]

Фото:
  [Главное фото] [+ Добавить ещё]
  (drag & drop или кнопка выбора файла)

[Сохранить]  [Удалить]
```

### Защита admin-панели

Простая защита через middleware — проверка секретного пароля в cookies:

```
/admin → если нет cookie admin_token → редирект на /admin/login
Логин: простая форма с паролем
Пароль хранится в .env.local как ADMIN_PASSWORD
```

## 4.5 Оптимизация фото

Используем встроенную оптимизацию Next.js:

```tsx
import Image from 'next/image'

<Image
  src={product.mainImage}
  alt={product.name}
  width={600}
  height={750}
  className="object-cover"
  priority={isHero}
/>
```

- Автоматически конвертирует в WebP
- Lazy loading для карточек
- Priority для hero-фото (загружается первым)
