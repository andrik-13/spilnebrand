# SPIL'NE — План разработки

## Стек

```
Next.js 14 (App Router)
TypeScript
Tailwind CSS
Supabase (DB + Storage)
Telegraf.js (Telegram бот)
Vercel (хостинг)
```

## Пошаговый план

```
Этап 0 — ПОДГОТОВКА (1-2 дня) — делаешь ты
├── Собираешь все фото товаров в одну папку
├── Конвертируешь логотип в SVG (из PSD)
└── Создаёшь аккаунт Supabase (бесплатно)

Этап 1 — ДИЗАЙН В FIGMA (3-5 дней) — делаешь ты по брифу (figma.md)
├── Design System (цвета, шрифты, компоненты)
├── Desktop: Главная, Каталог, Товар
├── Mobile: те же страницы
└── ✅ Показываешь мне — корректируем

Этап 2 — РАЗРАБОТКА: Проект (1 день) — делаю я
├── Создаю Next.js 14 + TypeScript + Tailwind
├── Настраиваю шрифты и цвета по Design System
└── Структура папок и роутинг

Этап 3 — РАЗРАБОТКА: Дизайн → Код (3-5 дней) — делаю я
├── Header + Footer
├── Главная страница
├── Каталог
└── Страница товара

Этап 4 — РАЗРАБОТКА: Backend (2-3 дня) — делаю я
├── Supabase: таблица товаров, хранилище фото
├── API endpoints (список товаров, один товар)
└── Admin-страница (добавить/редактировать товар)

Этап 5 — TELEGRAM БОТ (2 дня) — делаю я
├── Бот принимает заказ (товар + размер)
├── Спрашивает имя, телефон, город, отделение НП/Укрпочта
└── Отправляет заявку тебе + подтверждение клиенту

Этап 6 — КОНТЕНТ (1-2 дня) — вместе
├── Добавляем 21 товар в базу
├── Пишем тексты на UA + EN
└── SEO мета-теги

Этап 7 — ЗАПУСК (1 день)
├── Деплой на Vercel
├── Настройка домена
└── Финальное тестирование
```

## Информация о бренде

- **Название:** SPIL'NE (с апострофом, в домене без него)
- **Локация:** Украина
- **Продукт:** Базовая женская одежда (футболки, юбки, штаны, спортивные костюмы)
- **SKU:** 21 позиция
- **Размеры:** S, M
- **Языки сайта:** Украинский + Английский
- **Доставка:** Новая Почта, Укрпочта
- **Оплата:** через Telegram бот

## Структура страниц

### Главная
- Hero (полный экран, фото на модели, слоган)
- Новая коллекция (4 товара)
- О бренде (текст + фото)
- Instagram / Лукбук галерея
- Footer

### Каталог
- Фильтр по категории (Усі / Tops / Bottoms / Sets)
- Сетка товаров (3 колонки desktop, 2 mobile)

### Страница товара
- Галерея фото
- Название, цена, размеры
- Кнопка "ЗАМОВИТИ У TELEGRAM"
- Состав и уход (аккордеон)
- Доставка (аккордеон)
