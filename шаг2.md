## 2.4 Переменные окружения (.env.local)

## 2.6 Мультиязычность

Используем библиотеку `next-intl`.

**Как это работает:**
- URL `/ua/catalog` — украинская версия
- URL `/en/catalog` — английская версия
- При первом заходе — автоопределение языка браузера
- Переключатель UA|EN в хедере

**Файлы переводов:**
```json
// messages/ua.json
{
  "nav": {
    "catalog": "Каталог",
    "about": "Про бренд"
  },
  "product": {
    "size": "Розмір",
    "order": "Замовити у Telegram",
    "contact": "Ми зв'яжемося з вами у Telegram"
  }
}
```

## 2.7 Как запустить локально (для тебя)

После того как я создам проект:

```bash
# 1. Перейти в папку проекта
cd spilne-shop

# 2. Установить зависимости
npm install

# 3. Создать .env.local и заполнить своими ключами Supabase
cp .env.example .env.local

# 4. Запустить
npm run dev

# 5. Открыть в браузере
# http://localhost:3000
```
