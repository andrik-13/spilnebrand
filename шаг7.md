## 7.1 Деплой на Vercel

Vercel — это хостинг специально для Next.js. Бесплатный план полностью покрывает наши нужды.

### Подключение репозитория

1. Зайди на https://vercel.com → Login with GitHub
2. New Project → выбери репозиторий `spilne-shop`
3. Framework: Next.js (определится автоматически)
4. Нажми Deploy

Vercel сам соберёт и задеплоит проект. Первый деплой занимает ~2 минуты.

После деплоя сайт будет доступен по адресу:
```
https://spilne-shop.vercel.app
```

### Переменные окружения на Vercel

Все переменные из `.env.local` нужно добавить в Vercel:

1. Vercel → проект → Settings → Environment Variables
2. Добавь каждую переменную:
   ```
   NEXT_PUBLIC_SUPABASE_URL
   NEXT_PUBLIC_SUPABASE_ANON_KEY
   TELEGRAM_BOT_TOKEN
   TELEGRAM_ADMIN_CHAT_ID
   ADMIN_PASSWORD
   ```
3. Выбери: Production + Preview + Development
4. Save

После этого сделай Redeploy (Deployments → трёхточечное меню → Redeploy).

## 7.2 Настройка домена

### Если домен уже куплен

1. Vercel → проект → Settings → Domains
2. Add Domain → введи `spilne.ua` (или другой)
3. Vercel покажет DNS записи которые нужно добавить:
   ```
   Type: A
   Name: @
   Value: 76.76.21.21
   
   Type: CNAME
   Name: www
   Value: cname.vercel-dns.com
   ```
4. Зайди к своему регистратору домена (nic.ua, namecheap и т.п.)
5. В DNS-настройках добавь эти записи
6. Подождать 15-60 минут — домен заработает

### Если домена ещё нет

**Купить на nic.ua** (для .ua домена):
- Нужен украинский паспорт для домена .ua
- Стоимость: ~400-600₴ в год

**Альтернатива на старте:** использовать `spilne.vercel.app` — бесплатно и без настроек.

## 7.3 Настройка Telegram webhook на production

После деплоя бот должен слушать webhook на реальном домене.

Выполни один раз (я сделаю):
```
https://api.telegram.org/bot{TOKEN}/setWebhook
  ?url=https://spilne.ua/api/telegram/webhook
```

Проверить что webhook установлен:
```
https://api.telegram.org/bot{TOKEN}/getWebhookInfo
```

2. Проверь OG-превью: https://opengraph.xyz

## 7.5 После запуска

### Google Search Console
1. Зарегистрируйся на https://search.google.com/search-console
2. Добавь домен
3. Подтверди владение (Vercel позволит добавить DNS-запись)
4. Отправь sitemap: `https://spilne.ua/sitemap.xml`

Это нужно чтобы Google начал индексировать сайт.

### Google Analytics (опционально)
Если хочешь отслеживать посещаемость:
1. https://analytics.google.com → новый аккаунт
2. Получи Measurement ID: `G-XXXXXXXXXX`
3. Добавим код на сайт
