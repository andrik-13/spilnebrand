## 0.1 Логотип → SVG

Логотип сейчас есть в JPG и PSD. Нужен чистый SVG.

**Как сделать:**
1. Открой PSD в Adobe Illustrator (или Figma → File → Place image)
2. Если логотип — просто текст в Book Antiqua: набери "SPIL'NE" в Figma, выбери шрифт, выровняй как в оригинале
3. Выдели → Right click → Outline stroke (если есть)
4. Export → SVG (не "SVG compressed")
5. Проверь: открой SVG в браузере, должен выглядеть чисто

**Что нужно на выходе:**
- `logo.svg` — основной логотип (тёмный, для белого фона)
- `logo-white.svg` — белая версия (для тёмного footer и hero)
- `logo-mark.svg` — если есть иконка/монограмма отдельно

## 0.2 Фото товаров

**Что собрать:**
- Все фото на моделях в одну папку
- Формат: JPG или WebP
- Минимальное разрешение: 1200x1600px (пропорция 3:4)
- Называть файлы понятно: `tshirt-white-model-1.jpg`, `skirt-beige-model-1.jpg`

**Что нужно сделать (фото без моделей):**
- Отдельные фото на плоской поверхности (flatlay) или на вешалке — для карточек товара
- Минимум 2-3 фото на каждую позицию (разные ракурсы)
- Белый или кремовый фон — в тему бренду

**Структура папки:**
```
/photos
  /tshirt-white
    model-1.jpg
    model-2.jpg
    flat-1.jpg
  /skirt-beige
    model-1.jpg
    flat-1.jpg
  ...
```
