## 3.5 Каталог

**Файл:** `app/[locale]/catalog/page.tsx`

**Компоненты:**
- `CategoryFilter` — горизонтальный фильтр: Усі / Tops / Bottoms / Sets
- `ProductGrid` — сетка карточек
- `ProductCard` — карточка товара

**Логика фильтра:**
- Фильтр меняет URL параметр: `/catalog?category=tops`
- При изменении категории — перезагрузка карточек
- Активная категория подчёркнута

**Карточка товара:**
```tsx
<ProductCard
  image="/photos/tshirt-white/model-1.jpg"
  title="Футболка базова"
  price={890}
  sizes={['S', 'M']}
  slug="tshirt-white"
/>
```

При клике на карточку → переход на страницу товара

## 3.6 Страница товара

**Файл:** `app/[locale]/catalog/[slug]/page.tsx`

**Desktop layout (две колонки):**

```
Левая (60%):
  Большое фото (пропорция 3:4)
  Миниатюры снизу (3 шт)

Правая (40%, sticky):
  Категория (small, muted)
  Название (H2)
  Цена (H3)
  ---линия---
  РОЗМІР
  [S] [M]
  [ЗАМОВИТИ У TELEGRAM]
  "Ми зв'яжемося з вами у Telegram"
  ---линия---
  [Склад та догляд] ↕
  [Доставка] ↕
```

**Mobile:**
- Галерея фото (swipe)
- Весь контент снизу
- Кнопка "ЗАМОВИТИ" — fixed внизу экрана

**Кнопка "ЗАМОВИТИ У TELEGRAM":**
- Собирает данные: название товара + размер
- Формирует ссылку: `https://t.me/spilne_bot?start=tshirt-white-S`
- Открывает Telegram с предзаполненным сообщением

## 3.7 Компоненты UI

```tsx
<SizeSelector
  sizes={['S', 'M']}
  selected={selectedSize}
  onChange={setSelectedSize}
/>
```

### ProductGallery
```tsx
<ProductGallery
  images={product.images}
/>
// Desktop: большое фото + миниатюры
// Mobile: swipe-галерея
```

### Accordion
```tsx
<Accordion title="Склад та догляд">
  100% бавовна. Машинне прання 30°
</Accordion>
```

## 3.8 Анимации и интерактивность

Минимально, без перебора:

| Элемент | Анимация |
|---|---|
| Карточка товара | hover: фото темнеет (transition 200ms) |
| Кнопки | hover: цвет темнее (transition 150ms) |
| Мобильное меню | slide-in (300ms ease) |
| Аккордеон | height expand (200ms ease) |
| Переключение размера | instant (без анимации) |

Используем Tailwind `transition` классы — никаких лишних библиотек.

## 3.9 SEO и мета-теги

Для каждой страницы прописываем:

```tsx
// app/[locale]/catalog/[slug]/page.tsx
export async function generateMetadata({ params }) {
  const product = await getProduct(params.slug)
  return {
    title: `${product.name} | SPIL'NE`,
    description: product.description,
    openGraph: {
      images: [product.images[0]],
    },
  }
}
```
