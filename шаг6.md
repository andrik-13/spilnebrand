## 6.1 Загрузка товаров в базу

К этому шагу у тебя уже есть таблица товаров из Шага 0.  
Загружаем всё через admin-панель (`/admin`).

**Для каждого из 21 товара:**

1. Открой `/admin/products/new`
2. Заполни поля:
   - Название UA + EN
   - Slug (автозаполнится, но проверь)
   - Категория (tops / bottoms / sets)
   - Цена
   - Размеры (S, M)
   - Описание UA + EN
   - Состав ткани UA + EN
   - Флаги: Новинка / Активен
3. Загрузи фото (главное + доп. ракурсы)
4. Нажми "Сохранить"

**Совет:** начни с 4-6 товаров, которые покажешь на главной как "Нова колекція" — отметь их флагом "Новинка".

## 6.2 Фото — требования перед загрузкой

Перед загрузкой обработай фото:

| Параметр | Требование |
|---|---|
| Формат | JPG или WebP |
| Разрешение | минимум 1200×1600px (пропорция 3:4) |
| Вес | до 2MB (оптимизируй через squoosh.app) |
| Фон | белый или кремовый |
| Цвета | без сильной коррекции, натуральные |

**Инструменты для оптимизации:**
- https://squoosh.app — сжатие без потери качества
- https://www.birme.net — пакетный resize

## 6.4 SEO — мета-теги

Для каждой страницы нужны:

### Главная
```
Title: SPIL'NE — Базовий жіночий одяг в Україні
Description: Якісний базовий одяг для жінок. Футболки, спідниці, штани та спортивні костюми. Доставка Новою Поштою по всій Україні.
```

### Каталог
```
Title: Каталог | SPIL'NE
Description: Базовий жіночий одяг — футболки, спідниці, штани, спортивні костюми. Розміри S та M. Замовлення через Telegram.
```

### Товар (шаблон)
```
Title: {Назва товару} | SPIL'NE
Description: {Назва товару} — {короткий опис}. Ціна {ціна} ₴. Доставка по Україні.
```

### OG-теги (для соцсетей)
```
og:image — главное фото товара или брендовое фото для главной
og:title — то же что Title
og:description — то же что Description
```

## 6.5 Иконка сайта (favicon)

Нужна иконка для браузерной вкладки:

- `favicon.ico` — 32×32px
- `icon.png` — 192×192px (для мобильных)
- `apple-icon.png` — 180×180px (для iOS)

Можно сделать из логотипа SPIL'NE — просто монограмму "S" или "SP".  
Инструмент: https://favicon.io
